# Sales Analytics with SQL, Time Series & Statistical Testing

Sales data analysis project combining SQL, exploratory data analysis, time series forecasting and statistical hypothesis testing.

## Project Overview

The project explores a retail sales dataset using several analytical approaches:

* SQL analysis with SQLite
* Time series analysis and forecasting
* Statistical hypothesis testing
* Data visualization

The goal is to practice a complete analytical workflow from querying raw data to generating business insights.

## Technologies:

* Python
* Pandas
* SQLite
* Matplotlib
* Statsmodels
* SciPy

## Dataset  
The dataset is included in this repository for demonstration and educational purposes.

## Project Components

### SQL Analytics

* Imported transactional data into SQLite.
* Wrote analytical SQL queries using `GROUP BY`, `ORDER BY`, `HAVING`, aggregate functions and date functions.
* Analyzed sales by category, region and month.
* Identified top-performing categories and regions.

### Time Series Analysis

* Aggregated daily and monthly sales.
* Applied rolling averages for trend smoothing.
* Performed seasonal decomposition.
* Built forecasting models using SARIMAX.

### Statistical Testing

* Performed independent two-sample t-tests.
* Evaluated statistical significance using p-values.
* Interpreted hypothesis testing results in a business context.

### Visualization

* Created analytical dashboards using Matplotlib.
* Visualized sales trends, regional performance, shipping methods and delivery times.
